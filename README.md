# Daylight Docs

Documentation website for the Daylight Rails monitoring gem.

## Overview

This is the official documentation site for Daylight, a Rails engine for application monitoring. Built with Jekyll and hosted on Cloudflare Pages.

## About Daylight

Daylight is a Rails engine that provides:

- **Error Tracking**: Automatic exception capture with fingerprinting and deduplication
- **Performance Monitoring**: Request timing, slow query detection, and N+1 identification
- **AI-Powered Insights**: Intelligent diagnostics and recommendations
- **SQLite-Backed**: No external services required

## Tech Stack

- **Static Site Generator**: Jekyll
- **Styling**: Custom CSS
- **Hosting**: Cloudflare Pages
- **Ruby**: 3.x

## Local Development

1. Install dependencies:
   ```bash
   bundle install
   ```

2. Start development server:
   ```bash
   bundle exec jekyll serve
   ```

3. Build for production:
   ```bash
   bundle exec jekyll build
   ```

## Documentation Structure

- `/getting-started` - Installation and setup guide
- `/error-tracking` - Error tracking documentation
- `/request-monitoring` - Performance monitoring guide
- `/_guides` - Additional guides and tutorials
- `/_reference` - API reference documentation
